# EmojiSpam Help

Prevent users from spamming emojis.

# emojispam
 - Usage: `[p]emojispam`
 - Restricted to: `ADMIN`
 - Checks: `server_only`

Manage emoji spam settings.

## emojispam limit
 - Usage: `[p]emojispam limit <limit>`

Set the emoji limit.

## emojispam toggle
 - Usage: `[p]emojispam toggle`

Toggle emoji spam prevention.

## emojispam logchannel
 - Usage: `[p]emojispam logchannel <channel>`

Set the log channel.

## emojispam togglemessage
 - Usage: `[p]emojispam togglemessage`

Toggle the message to send when a user spams emojis.

## emojispam message
 - Usage: `[p]emojispam message <message>`

Set the message to send when a user spams emojis.

## emojispam timeout
 - Usage: `[p]emojispam timeout <seconds>`

Set the timeout for the user.
